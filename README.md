# BinancePriceBar

![Touchbar](https://github.com/phhai1710/BinancePriceBar/blob/main/Resources/touchbar.gif?raw=true)

Cryptocurrency prices on MacBook Touch Bar, <b>running real time</b>.

The app runs in background and can be accessed anywhere by tapping the `â‚¿` icon on the Touch Bar Control Strip.

## Features

- [x] Add new coin pair.
- [x] Trigger alert.
- [x] 24h volume.
- [x] Chart.
- [x] Change chart interval.
- [x] Start at login.
- [x] Show Esc.
- [x] RSI 14 indicator.
- [ ] Show Binance account's data.
- [ ] ...
